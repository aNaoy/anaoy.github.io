**Mise à jour Windows 365 : accès aux Cloud PC perturbé**

Une mise à jour récente du service Windows 365, conçue pour améliorer la sécurité, a involontairement entraîné des problèmes d'accès aux sessions Cloud PC pour les utilisateurs. Microsoft a constaté une augmentation des tentatives de connexion échouées et a identifié la mise à jour comme la cause première.

Par ailleurs, une mise à jour de sécurité Windows publiée en janvier 2026 (KB5074109) est également signalée comme causant des erreurs d'authentification et des échecs de connexion lors de l'accès aux sessions via le Bureau à distance pour Azure Virtual Desktop et Windows 365. Cette dernière affecte notamment Windows 11 25H2, 24H2, ainsi que Windows Server 2025, 2022 et 2019.

**Points Clés :**

*   Une mise à jour de Windows 365 a bloqué l'accès aux Cloud PC.
*   Une mise à jour de sécurité Windows (KB5074109) cause des problèmes similaires avec Azure Virtual Desktop et Windows 365.

**Vulnérabilités :**

*   Aucune vulnérabilité spécifique avec un identifiant CVE n'est explicitement mentionnée dans l'article. Les problèmes découlent d'une "mise à jour" ou d'un "problème connu" plutôt que d'une faille de sécurité exploitée.

**Recommandations :**

*   **Solutions de contournement temporaires :** Accéder aux Cloud PC via le client Web Windows App à l'adresse windows.cloud.microsoft ou utiliser le client Bureau à distance pour Windows afin de se connecter à Azure Virtual Desktop.
*   Microsoft travaille à l'identification de la cause sous-jacente et à la mise en place d'une solution permanente pour corriger les problèmes liés à la mise à jour Windows 365.
*   La mise à jour de sécurité KB5074109 est reconnue comme un problème connu affectant plusieurs versions de Windows et Windows Server.
