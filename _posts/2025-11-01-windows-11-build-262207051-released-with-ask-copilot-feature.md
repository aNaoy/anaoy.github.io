**Nouvelles Fonctionnalités et Améliorations pour Windows 11**

La dernière mise à jour de Windows 11, la build 26220.7051, introduit plusieurs nouveautés pour les testeurs du programme Insider, avec un déploiement prévu pour tous dans les semaines à venir.

Parmi les ajouts notables, "Ask Copilot" est désormais accessible directement depuis la barre des tâches. Cette nouvelle interface permet d'interagir avec l'IA pour rechercher des informations sur Internet, des fichiers locaux et utiliser des requêtes vocales ou textuelles. Cette fonctionnalité devrait à terme remplacer l'interface de recherche Windows actuelle.

Une nouvelle expérience "Full Screen Experience" (FSE) pour les applications Xbox est également déployée, optimisant les performances sur les appareils portables et permettant une immersion complète dans les jeux. Cette option peut être activée dans les paramètres sous "Jeux".

L'assistance audio partagée pour les appareils Bluetooth, basée sur la technologie LE Audio, permet désormais de diffuser un flux audio vers deux appareils simultanément. Cette fonctionnalité, initialement en préversion, peut être activée via les paramètres rapides.

Enfin, les PC équipés de processeurs ARM bénéficient d'une amélioration des performances grâce à une meilleure prise en charge de l'émulation des applications 64 bits x86 (x64).

**Points Clés :**

*   Intégration de "Ask Copilot" dans la barre des tâches pour une recherche IA simplifiée.
*   Nouvelle expérience plein écran optimisée pour les jeux Xbox sur appareils portables.
*   Support pour le partage audio sur deux appareils Bluetooth simultanément.
*   Amélioration des performances pour les PC ARM via une meilleure émulation x64.

**Vulnérabilités :**

Aucune vulnérabilité spécifique ou CVE n'est mentionnée dans cet article concernant cette mise à jour.

**Recommandations :**

*   Les utilisateurs intéressés par ces nouvelles fonctionnalités peuvent rejoindre le programme Windows Insider (canaux Beta ou Dev) pour les essayer dès maintenant.
*   Les fonctionnalités seront disponibles pour tous les utilisateurs dans les semaines suivant la publication de la mise à jour.
*   Pour activer "Ask Copilot" : Paramètres > Personnalisation > Barre des tâches > Ask Copilot.
*   Pour activer la "Full Screen Experience" Xbox : Paramètres > Jeux > Full screen experience.
*   Pour activer l'audio partagé Bluetooth : Activer le raccourci "Shared audio (preview)" dans les paramètres rapides.
