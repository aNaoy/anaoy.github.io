**Nouvelles Fonctionnalités Révélées pour ChatGPT**

Des tests internes chez OpenAI laissent présager l'arrivée prochaine de plusieurs améliorations significatives pour ChatGPT, attendues dans les semaines à venir sur la version web.

Parmi les nouveautés identifiées, on trouve :

*   **Fonctionnalité "Salute" :** Ce nouveau module permettrait la création de tâches à partir de fichiers téléchargés, avec un suivi de leur progression.
*   **Optimisation des résultats locaux :** Un indicateur "is model preferred" suggère que ChatGPT pourrait bientôt sélectionner un modèle optimisé pour les recherches locales, notamment pour les entreprises, restaurants et hôtels, améliorant ainsi les résultats affichés dans les widgets de cartes professionnelles.
*   **Tunnel sécurisé pour serveurs MCP :** Une nouvelle connexion sécurisée est en cours de développement pour les serveurs MCP, utilisant un tunnel client hébergé par le client et une connexion HTTPS sortante uniquement. Cela simplifie l'intégration en évitant les modifications des pare-feux entrants.
*   **Édition de blocs de code et de mathématiques :** Des blocs de code et de mathématiques éditables directement dans l'interface utilisateur sont en cours de test. Cette fonctionnalité fait écho à l'ajout récent de "blocs de formatage" offrant une barre d'outils d'édition pour le texte mis en surbrillance dans les zones de texte enrichi de ChatGPT.

Ces développements indiquent une volonté d'enrichir les capacités de ChatGPT en matière de gestion de tâches, d'amélioration de la pertinence des informations locales et d'ergonomie pour le contenu technique et mathématique.
