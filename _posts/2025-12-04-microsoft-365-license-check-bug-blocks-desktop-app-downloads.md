**Problème de Téléchargement des Applications Microsoft 365**

Un dysfonctionnement au sein de Microsoft 365 empêche actuellement les utilisateurs de télécharger les applications de bureau depuis la page d'accueil de la plateforme. Ce problème, affectant le processus de vérification des licences suite à une mise à jour récente, est signalé depuis le 2 novembre. Microsoft a identifié la cause comme étant un "problème de code" dans une "mise à jour de service" qui interfère avec la vérification des licences. L'entreprise travaille activement à un correctif et teste actuellement sa solution. Une mise à jour sur l'avancement est attendue.

*   **Points Clés:**
    *   Impossibilité de télécharger les applications de bureau Microsoft 365.
    *   Cause identifiée : bug dans la vérification des licences dû à une mise à jour de service.
    *   Le problème affecte potentiellement tous les utilisateurs tentant de télécharger les applications.
    *   Microsoft est en phase de test d'un correctif.

*   **Vulnérabilités:**
    *   Aucune vulnérabilité spécifique avec un identifiant CVE n'est mentionnée dans l'article. Le problème est qualifié de "bug" ou "dysfonctionnement" impactant le service.

*   **Recommandations:**
    *   Attendre le déploiement du correctif par Microsoft.
    *   Les utilisateurs sont invités à consulter les mises à jour officielles de Microsoft pour l'état du problème.
