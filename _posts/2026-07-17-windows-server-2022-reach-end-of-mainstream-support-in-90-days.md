### Fin du support standard pour Windows Server 2022

Windows Server 2022 atteindra la fin de son support standard (mainstream support) le 13 octobre 2026. À cette date, le système basculera en support étendu, garantissant la réception de mises à jour de sécurité mensuelles gratuites jusqu'au 14 octobre 2031.

**Points clés :**
*   **Transition de cycle de vie :** Le passage au support étendu n'entraîne pas de rupture immédiate des correctifs de sécurité, mais marque la fin des nouvelles fonctionnalités et des demandes de support non liées à la sécurité.
*   **Hotpatching :** Pour les versions *Datacenter: Azure Edition*, la fonctionnalité de hotpatching est exceptionnellement étendue jusqu'en octobre 2027.
*   **Alternative recommandée :** Microsoft préconise une migration vers Windows Server 2025, disponible depuis novembre 2024, pour bénéficier d'un support standard complet jusqu'en 2029.

**Vulnérabilités :**
*   Aucune CVE spécifique n'est mentionnée dans l'article. Il s'agit d'une annonce de gestion de cycle de vie produit et non d'une alerte de vulnérabilité active.

**Recommandations :**
*   **Anticipation :** Évaluer les options de mise à niveau et débuter les tests de déploiement de Windows Server 2025 dès maintenant pour garantir une transition fluide.
*   **Essais :** Utiliser la version d'évaluation gratuite de 180 jours disponible via le *Microsoft Evaluation Center* pour tester l'environnement avant la bascule définitive.
*   **Planification :** Consulter régulièrement le Lifecycle Policy search tool de Microsoft pour suivre les dates de fin de support des autres composants de l'infrastructure.
