### Problèmes d'installation de Microsoft Office sur Windows 365

Certains utilisateurs de Windows 365 rencontrent des difficultés pour télécharger et installer la suite Microsoft Office. Ce dysfonctionnement fait suite à une modification de configuration introduite lors d'une récente mise à jour du service (référencée sous l'identifiant WP1309017).

**Points clés :**
* **Origine :** Une erreur de configuration dans une mise à jour déployée par Microsoft.
* **Impact :** Les utilisateurs de Windows 365 (Enterprise et Business) peuvent échouer lors de l'installation ou du téléchargement d'Office.
* **État actuel :** Microsoft développe un correctif, dont le déploiement est prévu pour le vendredi suivant l'incident. Le problème est classé comme un avis de service à impact limité.
* **Vulnérabilités :** Aucune vulnérabilité de sécurité (CVE) n'est associée à cet incident technique.

**Recommandations :**
* En attendant le correctif officiel, les utilisateurs concernés peuvent contourner le problème en téléchargeant manuellement la suite logicielle directement via le [portail officiel Microsoft 365](https://www.microsoft.com/en-us/microsoft-365/download-office).
