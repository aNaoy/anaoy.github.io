**Extension des capacités de restauration Windows pour les entreprises**

Microsoft a élargi la fonctionnalité de restauration de Windows pour inclure davantage de scénarios d'entreprise. Cette fonctionnalité, intégrée à Windows Backup for Organizations, permet aux utilisateurs de restaurer leurs paramètres personnels et la liste des applications Microsoft Store sur un nouvel appareil ou un appareil réimaginé lors de leur première connexion.

Auparavant limitée aux appareils joints à Microsoft Entra, cette option est désormais disponible pour les environnements hybrides, les configurations multi-utilisateurs et les Windows 365 Cloud PCs. Cela facilite la migration vers Windows 11 et la gestion des renouvellements fréquents de matériel pour les organisations. Les administrateurs informatiques conservent le contrôle total sur l'application de cette fonctionnalité via des stratégies existantes gérées par Microsoft Intune ou Group Policy. La disponibilité générale a débuté avec les mises à jour Windows publiées le 24 février 2026.

**Points clés :**

*   Extension de la fonctionnalité de restauration des paramètres et des applications du Microsoft Store à un plus grand nombre d'appareils d'entreprise.
*   Intégration dans Windows Backup for Organizations pour simplifier les sauvegardes et les migrations vers Windows 11.
*   Permet la restauration lors de la première connexion, même si l'option a été manquée lors de la configuration initiale.
*   Prise en charge des environnements hybrides, des appareils multi-utilisateurs et des Windows 365 Cloud PCs.
*   Géré via des politiques existantes (Microsoft Intune, Group Policy), sans nécessiter de nouveaux outils pour les administrateurs.

**Vulnérabilités :**

Aucune vulnérabilité spécifique (CVE) n'est mentionnée dans cet article.

**Recommandations :**

*   Les organisations utilisant des environnements hybrides, des appareils multi-utilisateurs ou des Windows 365 Cloud PCs peuvent désormais bénéficier de la fonctionnalité de restauration des paramètres et des applications du Microsoft Store.
*   Les administrateurs informatiques devraient examiner et configurer les politiques de Windows Backup for Organizations pour déployer cette fonctionnalité selon leurs besoins.
*   Cette fonctionnalité est particulièrement utile pour les migrations vers Windows 11 et les cycles fréquents de renouvellement de matériel.
