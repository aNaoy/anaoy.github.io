### Améliorations IA pour Notepad et Paint

Microsoft intègre des fonctionnalités d'intelligence artificielle dans les dernières versions de Notepad et Paint, actuellement distribuées aux Insiders de Windows 11 dans les canaux Canary et Dev.

**Notepad :**

*   **Fonctionnalités IA :** Le bloc-notes propose désormais des options "Écrire", "Réécrire" et "Résumer" alimentées par l'IA. Les résultats de ces fonctions s'affichent par aperçus plus rapides, sans attendre la réponse complète. L'accès à ces outils IA nécessite une connexion avec un compte Microsoft.
*   **Support Markdown étendu :** Ajout de nouvelles fonctionnalités pour la syntaxe Markdown, incluant les listes imbriquées et le texte barré, utilisables via raccourcis clavier, la barre d'outils de formatage ou directement en syntaxe Markdown.
*   **Écran d'accueil :** Une nouvelle fenêtre d'accueil guide les utilisateurs sur les nouvelles fonctionnalités, accessible aussi via une icône de mégaphone.

**Paint :**

*   **Livre de coloriage :** Une nouvelle fonctionnalité "Livre de coloriage" génère des pages à colorier à partir de descriptions textuelles, proposant plusieurs options à intégrer sur la toile. Cette option est exclusive aux PC Copilot+ et requiert une authentification du compte Microsoft.
*   **Curseur de tolérance de remplissage :** Permet de contrôler la précision de l'outil Remplissage, offrant plus de flexibilité pour différents effets.

**Points Clés :**

*   Intégration de l'IA dans des applications Windows basiques pour améliorer l'expérience utilisateur.
*   Fonctionnalités IA disponibles gratuitement pour les utilisateurs de PC Copilot+ sous Windows 11.
*   Nécessité d'une connexion au compte Microsoft pour accéder aux outils IA dans Notepad et Paint.

**Vulnérabilités :**

Aucune vulnérabilité spécifique avec des identifiants CVE n'est mentionnée dans cet article. L'article se concentre sur les nouvelles fonctionnalités.

**Recommandations :**

*   Les utilisateurs souhaitant désactiver les nouvelles fonctionnalités IA peuvent le faire dans les paramètres ou désinstaller la version mise à jour de Notepad pour revenir à la version intégrée `notepad.exe`.
*   Microsoft encourage les utilisateurs à soumettre leurs commentaires via le Hub de commentaires Windows (catégorie Applications) pour identifier et corriger les bugs.
